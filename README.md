# Msgpackjs

This is an implementation of the MessagePack serialization format. Read the [MessagePack specification](https://github.com/msgpack/msgpack/blob/master/spec.md) and browse [their website](https://msgpack.org/) for more information.

I decided to write this MessagePack implementation for my moomoo.io script because moomoo.io uses MessagePack for their network codec. At the moment, my moomoo.io script uses a msgpack library I found on NPM, but I want to use my own libraries, since using libraries created by other people is skidding, so I decided to write this to not be a skid.
